# MiniNN

MiniNN is a (mini) neural network library. It is an Automatic Differentiation library with basic neural network code. 


# Implemented

## Layers
- Linear
- Dropout 
- LogSoftmax

## Activations
- ReLU
- Sigmoid
- Tanh

## Optimizers
- Stochastic Gradient Descent
- Adam

## Loss Functions
- CrossEntropyLoss
- MSELoss
